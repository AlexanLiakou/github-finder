
# GitHub Finder

A small github profile finder application. It consumes the [GitHub](https://github.com/) API.

<br>

## Tech Stack

**Main:** [Vite](https://vite.dev/), [React.js](https://react.dev/), [TypeScript.js](https://www.typescriptlang.org/), [Tanstack Query](https://tanstack.com/query/latest)

**Packages:** [React Icons](https://www.npmjs.com/package/react-icons), [Use Debounce](https://www.npmjs.com/package/use-debounce?activeTab=dependencies)

<br>

## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`VITE_GITHUB_API_URL=http://api.github.com`

<br>

## Demo

## Run Locally

Clone the project

```bash
  git clone https://github.com/AlexanLiakou/github-finder.git
```

Go to the project directory

```bash
  cd github-finder
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
